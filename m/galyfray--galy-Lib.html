galyLib

galyLib is a mindustry mod Lib currently WIP

Features : * BarUtils : * addDefaultHealth() : add the default health bar * addDefaultPower() : add the default power bar (for "normal block" block power block will come later) * addDefaultLiquid() : add the default liquid bar * addDefaultItem() : add the default item bar * addDefaultBars() : add all the default bars (liquid, item, health, power for normal block) * addBar() : allow you to simply add simple bars (like the health one more complex bars like power bars can't be set using this func) * you can see an exemple in the walls.js file * new block template : * ErraticMendingWall : a wall which randomly heal over random time with a heat system
